# Airline Expenses Website

This repository contains a simple website created to showcase graphs developed for a **university coursework report**.  
It is my **first web development project**, built using **HTML and CSS**.

The website presents **bar charts of U.S. airline expenses from 1995 to 2020**, making the data easier to explore visually.

---

## Project Overview

- Displays airline expense data using bar charts
- Focuses on clear visual presentation of historical data
- Designed as an introductory web development project

---

## Technologies Used

- HTML
- CSS

---

## Data Source

The data used for the graphs comes from the MIT Airline Data Project:  
http://web.mit.edu/airlinedata/www/Expenses&Related.html
